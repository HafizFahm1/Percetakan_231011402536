Deskripsi Project

PercetakanApp_231011402536 merupakan aplikasi desktop berbasis Java yang dibuat menggunakan NetBeans IDE dan MySQL. Aplikasi ini digunakan untuk membantu pengelolaan data pemesanan pada usaha percetakan, mulai dari proses login, input data pesanan, hingga pengelolaan informasi transaksi.

Project ini dibuat sebagai tugas mata kuliah Pemrograman 2 pada Program Studi Teknik Informatika Universitas Pamulang.

Fitur Aplikasi

* Login pengguna
* Menu Utama
* Input data pesanan percetakan
* Penyimpanan data ke database MySQL
* Koneksi database menggunakan JDBC
* Pengelolaan data pesanan

Teknologi yang Digunakan

* Java
* NetBeans IDE
* MySQL
* JDBC

Struktur Project

src/
├── koneksi/
│ └── Koneksi.java
├── view/
│ ├── LoginForm.java
│ ├── MenuUtama.java
│ └── FormOrder.java
└── TestKoneksi.java

Cara Menjalankan Aplikasi

1. Import database MySQL yang digunakan.
2. Buka project di NetBeans IDE.
3. Pastikan konfigurasi database pada file `Koneksi.java` sudah sesuai.
4. Jalankan file `LoginForm.java` sebagai main program.
5. Login ke aplikasi dan gunakan fitur yang tersedia.
